# BackToTheBase

[English](README.md) | [简体中文](README.zh-CN.md)

一个用于触发末影珍珠、让机器人返回基地的 Xinbot 插件。

## 功能
- 触发配置的按钮来发射末影珍珠
- 自动寻路并寻找安全站立点按下按钮
- JSON 配置简单易用

## 使用方法
1. 安装 [MovementSync](https://github.com/huangdihd/movementsync)。
2. 从 Releases 页面下载插件 JAR。
3. 将 JAR 放入 Xinbot 的 plugins 目录。
4. 放置一个可以触发末影珍珠的按钮，并确保机器人能够寻路到达该按钮附近的区域（3格范围内）。
5. 启动或重载 Xinbot 实例。
6. 编辑根目录生成的 `base_config.json` 文件，填写对应的玩家名称和他们对应的按钮方块坐标：

   ```json
   {
     "Steve": {
       "x": 100,
       "y": 64,
       "z": 200
     },
     "Alex": {
       "x": -50,
       "y": 70,
       "z": 30
     }
   }
   ```

   **配置文件说明：**
   - 这是一个 JSON 键值对映射。
   - `键 (Key)` (例如上面的 `"Steve"`): 代表可以发送指令的玩家的**游戏内 ID**。
   - `值 (Value)`: 是一个包含 `x`, `y`, `z` 的坐标对象，代表**按钮方块所在的确切坐标**。
   - 机器人会自动获取按钮的朝向，并在玩家私聊发送 `back` 指令时，自动走到按钮旁边能够按到的安全位置，然后瞄准并按下该按钮。
7. 再次重载 Xinbot 实例以应用配置。
8. 玩家在游戏内发送私聊消息 `back` 给机器人即可触发。
